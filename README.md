# `@kinolanka/eslint-config`

> [ESLint](https://eslint.org/) config for Nextjs projects.

## Usage

**Install**:

Add target version after #

```bash
$ npm i -D https://github.com/kinolanka/eslint-config-next.git#v1.0.0
```

**Edit `.eslintrc.json`**:

```json
{
  "extends": "@kinolanka/eslint-config-next"
}
```
